1.7 Python IDE (inferior python)

  • install the :lang by SPC f P and uncomment python in init.el
    • (python +lsp +poetry)
    • (company +tng)
    • pdb, ipdb, gdb, pdbpp
    • SPC f f -> for starting a new file
  • compile with SPC c c
  • start buffer
  • run-python
  • CTRL + C CTRL + C
  • imenu SPC s i
  • Issues with org mode:
    • config for tapping in REPL
    • in config.el; (map! :after comint :map comint-mode-map “TAB” nil [tab] nil)
